## 개요

수능 과학탐구 서비스는 학생과 관리자 관점을 분리하여 만들었습니다.

실제 서비스 출시를 생각하고 있기에 스마트폰 테블릿 데스크 탑을 이용하는 학생들의 편의성과

문제와 원칙을 업데이트하는 관리자의 편의성을 고려하였습니다.
